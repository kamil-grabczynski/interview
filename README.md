## Installation

```bash
$ npm install
```

## Running the app

#### 1. With docker:
```bash
docker compose up --build
docker compose up
```

#### 2. Without docker:

```bash
# development
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
